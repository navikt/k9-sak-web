ft-adapt
========

Her putter vi kode som brukast til å tilpasse seg komponenter eller anna kode frå ft-frontend-saksbehandling.

Dette kan for eksempel vere forskjell i typer frå ft-frontend-saksbehandling (basert på fp-sak DTOer) og k9-sak-web sinde typer
(basert på k9-sak, ung-sak, etc DTOer).

Hovedsakleg tenkt til kode som blir brukt i fleire andre moduler. Tilpassingskode som berre blir brukt i ein komponent/modul
kan plasserast der istaden.
